General Description

Este programa mide la distancia utilizando un sensor ultrasónico HC-SR04 y muestra la distancia medida en centímetros en un display LCD. Además, enciende diferentes LEDs según la distancia medida y permite controlar la medición y la visualización mediante teclas.
